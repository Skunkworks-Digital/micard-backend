MiCrad

Scope of Work & Project Overview

1. Introduction

MiCrad is a mobile-first platform transforming traditional business cards into interactive digital experiences. Leveraging NFC and QR code technology, it enables professionals to share dynamic profiles with a simple tap or scan, bridging physical and digital networking seamlessly.

2. Core Features & Modules

Profile Setup

Users create and customise profiles with contact details, social links, portfolios, videos, and branding.

QR/NFC Integration

Generates unique QR codes and NFC tag mappings for instant card activation and public profile linking.

Public Profile Access

Recipients tap or scan cards to view mobile-optimised public profiles instantly.

Analytics Dashboard

Track scans, taps, location data (optional with consent), and profile views in real-time.

Team/Organisation Cards (Phase 2)

Manage multiple cards under one organisation dashboard with standardised branding and team analytics.

3. User Flow

For Card Owners (App Users)

  • Sign Up / Log In via email, Google, or phone (Firebase Auth)
  • Onboarding Guide introducing MiCrad’s features
  • Profile Builder: input contact, socials, media, branding
  • Preview & Publish with QR/NFC code generation
  • Access analytics dashboard for engagement stats

For Recipients (Clients)

  • Scan QR Code or tap NFC card
  • Instant redirect to owner’s public profile page
  • Save contact options: vCard download, copy details, connect via social links
  • (Future) Lead capture form integration

4. Technical Requirements

  • Frontend: Flutter / React Native for cross-platform app
  • Firebase Authentication: email, Google, phone login
  • Firestore: profiles, QR/NFC mappings, analytics logs
  • Cloud Functions: dynamic link resolution, tap/scan logging, integrations
  • Hosting: Firebase Hosting for public profile pages
  • Storage: images, logos, media assets
  • Dynamic Links: seamless access across devices

5. Design Guidelines

  • Professional, minimalist UI with strong brand colours
  • Intuitive onboarding flows with visual progress
  • Mobile-first public profiles with fast load speeds
  • Accessible design adhering to WCAG standards

6. Privacy & Security

  • Data encryption in transit and at rest
  • Profile privacy controls (public, private, restricted)
  • Compliance with GDPR and POPIA data regulations
